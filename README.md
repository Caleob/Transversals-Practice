# Parallel Lines & Transversals Practice Tool

A responsive, interactive web application designed for high school geometry students to practice and master angle relationships in parallel line/transversal configurations.

## Features
- **Scaffolded Learning**: Progress through three stages for every problem:
  1. **Angle Naming**: Practice 3-point naming conventions.
  2. **Relationship Identification**: Identify angle pair types and congruence/supplementary properties.
  3. **Algebraic Solutions**: Solve for variables using geometric logic.
- **Dynamic Problem Generation**: Randomly generates problems across three difficulty levels (Values, Expressions, and Two Expressions).
- **Premium UI**: Modern design with responsive layouts and immediate visual feedback.
- **Performance Reporting**: Track progress and download a session summary report.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- SVG (for geometric diagrams)
